How Lazy Loading Works:
1. Browser sees <img loading="lazy" src="..." />
2. Image download deferred until near viewport
3. Skeleton loader shows placeholder
4. Image loads when user scrolls near it
5. Smooth fade-in transition applied
6. Result: Faster initial page load
